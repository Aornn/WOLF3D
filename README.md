# WOLF3D

Moteur de raycasting comme Wolfenstein3D en C

## Installation

```
git clone https://github.com/Aornn/WOLF3D.git WOLF3D
```

```
cd WOLF3D
```

```
make
```

```
./WOLF3D
```

## Controle

Fleches : mouvements + rotations

A et D : mouvements lateraux
